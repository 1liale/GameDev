Updates a Sprite\+Renderer\textquotesingle{}s Sprite reference on the Category and Label value it is set

If a Sprite\+Renderer is present in the same Game\+Object, the \mbox{\hyperlink{class_unity_engine_1_1_experimental_1_1_u2_d_1_1_animation_1_1_sprite_resolver}{Sprite\+Resolver}} will update the Sprite\+Renderer\textquotesingle{}s Sprite reference to the corresponding Sprite.
