Dialgoue Sequence Formatting Guide

Wanna type a sequence of dialogue to be properly rendered? Read this

Use % to seperate lines of dialogue. This will be used to distingish between bits of dialogue.

Dialogue component implemented by devJ is super big brain and will automatically be seperated into lines, so feel free to press enter wherever you want a new line!

To specify the character speaking, use this format:

Joe Crane@My name is Joe Crane!

Finally, make sure you name the file extension with .txt, or it won't work.

For a full working example, reference CharSelection.txt
