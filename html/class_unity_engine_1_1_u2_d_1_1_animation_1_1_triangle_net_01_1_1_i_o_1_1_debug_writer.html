Detailed Description

Writes a the current mesh into a text file.

File format:

num_nodes id_1 nx ny mark ... id_n nx ny mark

num_segs id_1 p1 p2 mark ... id_n p1 p2 mark

num_tris id_1 p1 p2 p3 n1 n2 n3 ... id_n p1 p2 p3 n1 n2 n3
